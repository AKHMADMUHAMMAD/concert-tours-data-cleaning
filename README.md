# Concert Tours Data Cleaning Project

A Python (Pandas) project for cleaning and transforming a messy dataset of global concert tours into an analysis-ready format.

## What was done
- Cleaned messy text data (symbols, special characters)
- Standardized column names
- Converted revenue fields to numeric format
- Extracted start and end years from tour duration
- Handled missing values

## Tools
- Python
- Pandas
- Jupyter Notebook

## Output
Clean dataset ready for data analysis and visualization.
